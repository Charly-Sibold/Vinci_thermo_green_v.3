Voici les différent fichiers qui faut dans votre lib :

jcommon-1.0.23.jar
jfreechart-chart-1.0.19.jar
mysql-connector-j-8.1.0.jar

Pour obtenir jcommon-1.0.23.jar et jfreechart-chart-1.0.19.jar , il faut allais chercher dans le premier prototype de thermo-green ou bien sur le site jar-download.com
Pour ce qui est de mysql-connector-j-8.1.0.jar allez le télécharger sur le site dev.mysql.com ( il faut le connector dans le https , c'est important)
Une fois les 3 fichiers télécharger , mettez-les dans votre fichier lib puis sur eclispe , fait-clique droit sur votre projet puis properties . une page s'ouvrira , vous devrez allez dans "Java Build Path" . Ensuite cliqué sur Add External JARs... , il aura plus qu'a choisir vos fichiers et tout sera bon .
